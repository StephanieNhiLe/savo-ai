# Setup Backend (Flask)

1. Navigate to Backend directory
```
cd backend
```
2. Setup Virtual environment
```
python3 -m venv venv
```
3. Acivate the Virtual environment
```
source venv/bin/activate
```
4. Install required packages and libraries
```
pip install -r requirements.txt
```
5. Run server using,
```
python -m flask run
```
(or simply)
```
python app.py
```
